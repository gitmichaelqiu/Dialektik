# Changelog

## v0.1.0

### Added

- App icon with dark mode and tinted variant support using Xcode Icon Composer `.icon` format for iOS, iPadOS, and macOS
- Room code display in re-host button for quick reconnection
- Preserved document selection across tab switches (restores last opened document, falls back to first accessible doc if permissions changed)
- Evidence card sharing over P2P mesh with team-based visibility and ownership tracking
- Layered asset catalog icon support (`folder.iconcomposer.icon` type) for both iOS and macOS
- Flutter SDK updated to 3.44.4
- Platform build guides in README for macOS, iOS/iPadOS, web, Windows, and Android
